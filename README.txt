-Detailed outline in TRMS Brief.pdf

-GUI and API for Tuition Reimbursement Management System
	-Employees can request reimbursement for various training programs
	-These requests are then reviewed and sent to the appropriate parties automatically

Notable features:
-Role based authorization
-User profile management (pictures & info)
-Persistent messaging between users
-Administrator role can-view, can NOT edit
-Automated timeline enforcer (send escalation emails if deadlines are met, etc)
-UUID driven relations in database

Tech stack:
-Java, MySQL, H2
-Angular, TS, HTTP
